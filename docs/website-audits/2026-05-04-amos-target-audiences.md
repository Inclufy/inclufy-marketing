## 6 Primaire Persona's

### 1. Solo-Marketer / MKB-eigenaar (NL/FR)
> "Ik heb geen marketingteam. Ik heb een product. Ik moet zichtbaar zijn op LinkedIn en Instagram zonder daar 10 uur per week aan te besteden."

| Eigenschap | Detail |
|---|---|
| **Bedrijfsgrootte** | 1-10 medewerkers |
| **Branche** | Consultants, SaaS-founders, e-commerce, lokale dienstverleners |
| **Pijn** | Tijd. Geen idee wat te posten. Inconsistent posten. |
| **Triggers** | "Ik weet dat ik moet posten maar..." |
| **Tier-fit** | **Starter (€29/mnd)** |
| **Killer-features** | AI Copilot, Multi-Channel Publishing, Smart Scheduling, Brand Kit |
| **Acquisition-kanaal** | LinkedIn organic, SEO ("AI marketing tool MKB"), Google Ads |
| **Conversie-CTA** | "Probeer 14 dagen gratis — geen creditcard" |

### 2. Marketing Manager bij MKB+ (NL)
> "Mijn team van 3 verspilt uren met handmatig posten over 5 kanalen. Ik wil ROI per campagne zien."

| Eigenschap | Detail |
|---|---|
| **Bedrijfsgrootte** | 25-150 medewerkers |
| **Branche** | B2B SaaS, professional services, retail-keten, event-organisaties |
| **Pijn** | Geen ROI-zicht. Tools-versnippering (Buffer + Hootsuite + Mailchimp + handmatig WhatsApp). Brand-drift. |
| **Triggers** | Quartely planning, budget-vraag van directie, tooling-consolidatie |
| **Tier-fit** | **Professioneel (€79/mnd)** |
| **Killer-features** | Campaign ROI Dashboard, Automation Workflows, Team RBAC, Opportunity Radar, Analytics Overview |
| **Acquisition-kanaal** | LinkedIn Ads, partner-events, vergelijkingssites (Capterra/G2) |
| **Conversie-CTA** | "Bereken je ROI met onze calculator" + demo-flow |

### 3. Event Marketer / Eventbureau (NL/MA)
> "Ik organiseer 12 evenementen per jaar. Bij elk event verzamel ik 200+ leads en verlies de helft in mijn Excel-soep."

| Eigenschap | Detail |
|---|---|
| **Bedrijfsgrootte** | 5-50 |
| **Branche** | Events, conferenties, hospitality, beurzen |
| **Pijn** | Lead-leakage post-event. Geen pre-event audience-targeting. Handmatig follow-up. |
| **Triggers** | Aankomende beurs, slecht ROI vorig event |
| **Tier-fit** | Professioneel + add-on Event-Intelligence |
| **Killer-features** | Event Intelligence, Smart Lead Scoring, Event Scanner, Lead Capture, Calendar Sync |
| **Acquisition-kanaal** | Event-platforms partnerships (Eventbrite, Bizzabo), trade-press |
| **Conversie-CTA** | "Try op je volgende event — gratis demo-license tot het event" |

### 4. Marketing-/Communicatie-Agency (NL/MA)
> "Ik beheer 8 klanten. Elke klant heeft eigen merk, social-accounts, deadlines. Mijn team verliest grip."

| Eigenschap | Detail |
|---|---|
| **Bedrijfsgrootte** | 5-30 medewerkers, beheert 5-50 klanten |
| **Branche** | Marketing/communicatie/creative agencies |
| **Pijn** | Multi-tenant chaos. Brand-mix-ups. Klant-rapportage handmatig. Pitch-decks per RFP from scratch. |
| **Triggers** | Klant-onboarding, RFP-deadlines, kwartaal-rapportages |
| **Tier-fit** | **Enterprise (€199/mnd)** — white-label, multi-org RBAC |
| **Killer-features** | RBAC per asset, Brand Kit per klant, Proposal/Pitch Module, Analytics multi-tenant, white-label |
| **Acquisition-kanaal** | Direct sales, agency-events, referral program |
| **Conversie-CTA** | "Plan een witlabel-demo voor je hele team" |

### 5. Accountancy / Boekhoudkantoor (MA primair, NL secondair)
> "Ik wil mijn klanten extra waarde leveren met digital marketing zonder zelf marketeer te worden."

| Eigenschap | Detail |
|---|---|
| **Bedrijfsgrootte** | 3-30 medewerkers, beheert 50-500 SME-klanten |
| **Branche** | Accountancy, fiscalisten, MKB-adviseurs |
| **Pijn** | Klanten vragen marketing-hulp. Geen capaciteit. Risico op klant-verlies aan agencies. |
| **Triggers** | Klant-uplift-services, verbreding pakket, Marokko-overheidsdigitaliseringseis |
| **Tier-fit** | **Enterprise multi-client** + accountancy-bundle (zie `docs/PRIJSLIJST_MAROKKO_ACCOUNTANCY.md`) |
| **Killer-features** | Multi-client RBAC, Brand Kit per klant, white-label rapportage, AI Copilot voor non-marketeers, FR/AR-talig |
| **Acquisition-kanaal** | Marokko: BMCE/Chaabi/Dar Group partnerships. NL: NOAB/SRA-events |
| **Conversie-CTA** | "Lever marketing-as-a-service zonder marketeer aan te nemen" + accountancy-bundle |

### 6. Enterprise In-House Marketing Team (NL/UAE)
> "Wij hebben 20 marketeers en gebruiken HubSpot/Salesforce. We willen AI-content + multi-region orkestratie."

| Eigenschap | Detail |
|---|---|
| **Bedrijfsgrootte** | 250+ medewerkers |
| **Branche** | Multinationals, financials, tech, retail-chains |
| **Pijn** | AI-tooling versnippering. Compliance & SSO-eisen. Multi-market localisatie. |
| **Triggers** | Marketing-stack-consolidatie, AI-mandate van CMO, compliance-audit |
| **Tier-fit** | **Enterprise + Custom** (white-label, SSO, dedicated AM, on-prem optie) |
| **Killer-features** | SSO, SAML, audit-logs, custom AI-modeltraining, multi-locale, dedicated AM |
| **Acquisition-kanaal** | Direct sales, RFP, partner-channels (UAE) |
| **Conversie-CTA** | "Plan een strategische demo met onze enterprise team" |

## Anti-personas (NIET targeten)

- **B2C-merken met >€10M ad-spend** → Inclufy is geen ad-platform; doorverwijzen
- **Single-channel-only specialists** (alleen TikTok, alleen LinkedIn) → multi-channel is ons verhaal
- **Pure media-buying agencies** → wij zijn organic + owned media first
- **Studenten / hobbyisten** → te lage ARPU, support-kosten te hoog
