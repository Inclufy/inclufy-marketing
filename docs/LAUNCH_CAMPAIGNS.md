## 1. De Inclufy Ecosystem Propositie

### Het Probleem

Groeiende bedrijven in de Benelux gebruiken gemiddeld 12-15 losse SaaS-tools. Het resultaat:

- **Marketing** werkt in Canva + Mailchimp + losse event-tools — geen connectie met projecten of financiele data
- **Projectteams** gebruiken WhatsApp + Excel + losse PM-tools — geen branded communicatie, geen financieel overzicht
- **Finance** zit vast aan Exact Online of dure SAP-licenties — geen AI, geen verbinding met operatie

Elk team koopt eigen tools. Niemand heeft het totaalbeeld.

### De Oplossing: Inclufy Platform

**Inclufy is het AI-powered business platform dat marketing, projecten en finance verbindt in een ecosystem.**

```
┌─────────────────────────────────────────────────────┐
│                  INCLUFY PLATFORM                    │
│                                                      │
│   ┌──────────┐   ┌──────────┐   ┌──────────────┐   │
│   │   AMOS   │   │ProjeXtPal│   │Inclufy Finance│   │
│   │  Event   │◄─►│ Project  │◄─►│   AI ERP     │   │
│   │Marketing │   │Management│   │              │   │
│   └──────────┘   └──────────┘   └──────────────┘   │
│          │              │               │            │
│          ▼              ▼               ▼            │
│   ┌─────────────────────────────────────────────┐   │
│   │         Gedeelde Data & Inzichten            │   │
│   │  Contacten · Projecten · Kosten · AI        │   │
│   └─────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────┘
```

### Ecosystem Waardepropositie per Stakeholder

| Stakeholder | Standalone waarde | Ecosystem meerwaarde |
|---|---|---|
| **Marketing Manager** | AMOS: branded content, NFC scanning, campaigns | + Projectvoortgang in campagnes, eventkosten direct in Finance |
| **Project Manager** | ProjeXtPal: foto-documentatie, taakbeheer, samenwerking | + Marketing-materiaal voor klantcommunicatie, projectbudget uit Finance |
| **CFO / Finance** | Finance: AI procurement, forecasting, governance | + Realtime projectkosten, marketing-ROI per event |
| **CEO / COO** | — | Totaalbeeld: van event > project > factuur in een dashboard |

## 4. Campagne 3: Cross-Sell Flywheel

### Het Flywheel Concept

```
         ┌──────────────────┐
         │  Klant start met │
         │    1 app (Land)   │
         └────────┬─────────┘
                  │
                  ▼
         ┌──────────────────┐
         │  Succes bereikt   │
         │  (Activate)       │
         └────────┬─────────┘
                  │
                  ▼
         ┌──────────────────┐
         │  Cross-sell       │
         │  trigger (Expand) │
         └────────┬─────────┘
                  │
                  ▼
         ┌──────────────────┐
         │  Ecosystem klant  │
         │  (Platform)       │
         └────────┬─────────┘
                  │
                  ▼
         ┌──────────────────┐
         │  Referral naar    │
         │  andere bedrijven │
         └──────────────────┘
```

### Cross-Sell Paden

**Pad 1: AMOS > ProjeXtPal**
- **Trigger:** AMOS-gebruiker maakt content voor een project/event
- **Moment:** Na 3e event in AMOS
- **Boodschap:** "Je evenementen zijn top. Maar hoe beheer je de projecten erachter? Ontdek ProjeXtPal."
- **Tactiek:** In-app notificatie + email met ProjeXtPal intro video
- **Incentive:** 1 maand gratis ProjeXtPal bij ecosystem upgrade

**Pad 2: ProjeXtPal > Inclufy Finance**
- **Trigger:** ProjeXtPal-gebruiker beheert 5+ projecten met budget-gerelateerde taken
- **Moment:** Na 2 maanden actief gebruik
- **Boodschap:** "Je projecten draaien goed. Maar heb je overzicht over de kosten? Inclufy Finance geeft je AI-powered financieel inzicht."
- **Tactiek:** In-app banner + gepersonaliseerde email van customer success
- **Incentive:** Gratis ROI-analyse van huidige projectkosten

**Pad 3: Inclufy Finance > AMOS**
- **Trigger:** Finance-klant organiseert bedrijfsevenement of klantenevent
- **Moment:** Wanneer "event" of "marketing" wordt vermeld in governance workflows
- **Boodschap:** "Je financien zijn op orde. Nu je merk ook. AMOS maakt elk event een merkbeleving."
- **Tactiek:** Customer success introductie + demo
- **Incentive:** Ecosystem korting (20% op bundle)

**Pad 4: Elke app > Ecosystem Bundle**
- **Trigger:** Klant gebruikt 2 apps succesvol
- **Moment:** Na 1 maand met 2e app
- **Boodschap:** "Je gebruikt al AMOS en ProjeXtPal. Voeg Finance toe en krijg het volledige plaatje — met 20% ecosysteemkorting."
- **Tactiek:** Account manager + landing page met vergelijking
- **Incentive:** 20% bundle korting + onboarding sessie

### Cross-Sell KPIs

| Metric | Target (6 maanden) |
|---|---|
| Single-app klanten die 2e app starten | 15% |
| 2-app klanten die 3e app toevoegen | 10% |
| Gemiddelde revenue per account (ARPA) stijging | +40% |
| Ecosystem NPS | > 50 |
